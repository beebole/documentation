# Documentation Structure

Use this context when creating or reviewing page structure. Languages, tabs, paths, and link rules are in CLAUDE.md — this file covers the page template and section ordering.

## Page structure template

1. **Frontmatter** — `title`, `description` (120-160 chars), `keywords` (3-8 terms)
2. **Introduction** — 2-4 sentences: what is this, when would I use it
3. **Core tasks** — Main actions with `##` subheadings and `<Steps>` components
4. **Configuration** — Settings and customization options
5. **Advanced / Edge cases** — Use `<Accordion>` for less common scenarios
6. **Troubleshooting** — Common problems and solutions
7. **FAQ section** — `<AccordionGroup>` with 3-5 Q&A pairs
8. **Related links** — Pointers to related pages
