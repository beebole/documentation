---
title: "Single Sign-On (SSO)"
description: "Set up single sign-on (SSO) for Beebole using SAML, OpenID Connect, Google, or Microsoft to centralize user authentication and simplify team access."
---

Beebole supports SSO via **Google**, **Microsoft**, and any **Custom OpenID Connect** provider (Okta, Auth0, Entra ID, and others). Full setup instructions, including domain linking, auto-provisioning, and SSO enforcement, are covered in the authentication guide.
