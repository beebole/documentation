## **Create an Absence Type**

Request:

```json
{
  "service": "absence.create",
  "absence": {
    "name": "myAbsence",
    "company": {"id": 233}
  }
}
```

Response:

```json
{
  "status": "ok",
  "id": 78
}
```

## **Get an Absence Type**

Request:

```json
{
  "service": "absence.get",
  "id": 78
}
```

Response:

```json
{
  "status" : "ok",
  "absence" : {
    "id" : 78,
    "name" : "myAbsence",
    "company" : {"id" : 233},
    "active" : true
  }
}
```
