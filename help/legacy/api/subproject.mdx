## **Create a Subproject**

Request:

```json
{
  "service": "subproject.create",
  "subproject": {
    "name": "Prototype",
    "project": {"id": 321}
  }
}
```

Response:

```json
{
  "status": "ok",
  "id": 765
}
```

## **Get a Subproject**

Request:

```json
{
  "service": "subproject.get",
  "id": 765
}
```

Response:

```json
{
  "status" : "ok",
  "subproject" : {
    "id" : 765,
    "name" : "Prototype",
    "project" : {"id" : 765},
    "active" : true
  }
}
```
