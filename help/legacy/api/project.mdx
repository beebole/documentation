## **Create a Project**

Request:

```json
{
  "service": "project.create",
  "project": {
    "name": "Development",
    "startDate": "2017-10-13", //optional, default 1st current month
    "description" : "some description ...", //optional
    "company": {"id": 233}
  }
}
```

Response:

```json
{
  "status": "ok",
  "id": 321
}
```

## **Get a Project**

Request:

```json
{
  "service": "project.get",
  "id": 321
}
```

Response:

```json
{
  "status" : "ok",
  "project" : {
    "id" : 321,
    "name" : "Development",
    "startDate": "2017-10-13",
    "description" : "some description ...",
    "company" : {"id" : 233},
    "subprojects" : {"count" : 2},
    "active" : true
  }
}
```
