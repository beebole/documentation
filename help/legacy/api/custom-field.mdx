## **List Custom Fields**

Request:

```json
{
  "service": "custom_field.list"
}
```

Response:

```json
{
  "customFields": [
    {
      "id":4641,
      "name":"Company address",
      "availableFor":"Company - Customer"
    }
  ]
}
```
