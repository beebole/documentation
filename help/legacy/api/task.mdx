## **Create a Task**

Request:

```json
{
  "service": "task.create",
  "task": {
    "name": "Meeting",
    "company": {"id": 233}
  }
}
```

Response:

```json
{
  "status": "ok",
  "id": 955
}
```

## **Get a Task**

Request:

```json
{
  "service": "task.get",
  "id": 955
}
```

Response:

```json
{
  "status" : "ok",
  "task" : {
    "id" : 955,
    "name" : "Meeting",
    "company" : {"id" : 233},
    "active" : true
  }
}
```

## **Update a Task**

Request:

```json
{
  "service" : "task.update",
  "task" : {
    "id" : 955,
    "name" : "newName"
  }
}
```

Response:

```json
{"status" : "ok"}
```
